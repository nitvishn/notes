How do we write numbers?

  • 1 - 10 : 一, 二, 三, 四, 五, 六, 七, 八, 九, 十

  • 11 - 19 : 十 + Number
    ex) 18 = 十八

  • 20 - 99 : Number + 十 + Number
    ex) 24 = 二十四
    ex) 50 = 五十

  • 101 - 999 : Number + 百 + Number + 十 + Number
    ex) 104 = 一百四
    ex) 729 = 七百二十九

  • 1001 - 9999 : Number + 千 + Number + 百 + Number + 十 + Number
    ex) 1009 = 一千九
    ex) 2405 = 二千四百五
    ex) 8793 = 八千七百九十三
